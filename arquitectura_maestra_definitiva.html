HAL & Low-Level Connectivity

I2C Protocol Internals

Low-level communication utilizing 16/32-bit hardware register maps for raw sensor access.

Register MapFunctional Purpose
0x00 (STATUS)Validation of RX/TX buffer states
0x01 (WRITE)Virtual register addressing bridge
0x04 (CONTROL)Integration time and gain parameters
0x14 - 0x28IEEE 754 Float spectral data output

Hardware Drivers

  • AS726X: SMBus with 50ms polling timeout for register stability.
  • MLX90640: Background threading for 2Hz asynchronous acquisition.
  • GPIO (RPi): Binary BCM control for high-precision LED strobing.
  • Mocks: Synthetic data injection with Gaussian noise for CI testing.
